# xml-modifier

Modifies XML documents based on fixed paths.

## Getting Started
* Navigate to this directory
```
cd FAKE_PATH/xml-modifier
```
* Run `index.js` in your terminal
```
node index.js
```
* Select the program you want to modify the XML file for


### Prerequisites

Have node installed
